# KPass_Currency
### web app that helps users to find best possible time for converting money from one currency to another, using historical currency rates

## Getting Start

`git clone https://github.com/niroshn/KPass_Currency.git`

### BackEnd (API)

`cd api`
`npm start`


### FrontEnd (Web)

`cd web`
`npm start`

## Used technologies
