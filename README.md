# KPass_Currency
## web app that helps users to find best possible time for converting money from one currency to another, using historical currency rates

## Getting Start

`git clone https://github.com/niroshn/KPass_Currency.git`

### BackEnd (API)

Run Development Environment 
- `cd api`
- `npm install`
- `npm start`

Run ESLint
- `npm run eslint`

Run Unit Test
- `npm test`
Run Unit Test with Test Coverage
- `npm test -- --coverage`

### FrontEnd (Web)

Run Development Environment 
- `cd web`
- `npm install`
- `npm start`

Run ESLint
- `npm run eslint`

Run Unit Test
- `npm test`
Run Unit Test with Test Coverage
- `npm test -- --coverage`
